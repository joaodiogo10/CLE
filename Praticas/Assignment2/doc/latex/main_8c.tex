\doxysubsection{Detailed Description}
Text processing in Portuguese program ~\newline
 

This program reads in succession several text files text\#.txt whose names are provided in the command line and prints a listing of total number of words, number of words beginning with a vowel and number of words ending with a consonant for each of the supplied files.
